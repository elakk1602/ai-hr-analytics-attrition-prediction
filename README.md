# AI-Powered HR Analytics & Attrition Prediction Dashboard

## Project Overview
This project is an AI-powered HR analytics dashboard developed using Power BI and Python to analyze employee attrition, workforce trends, salary insights, and department-wise performance. The dashboard helps organizations identify employee attrition patterns and supports data-driven HR decision-making.

---

## Features
- Employee Attrition Analysis
- AI-Based Attrition Prediction
- KPI Tracking Dashboard
- Department-wise Performance Insights
- Salary Trend Analysis
- Interactive Filters & Slicers
- Forecasting & Business Insights
- Data Cleaning & Transformation

---

## Tools & Technologies Used
- Power BI
- Python
- Pandas
- Machine Learning
- Excel
- DAX
- Power Query

---

## Machine Learning Model
Implemented a Random Forest Classifier to predict employee attrition based on:
- Age
- Monthly Income
- Years at Company
- Job Satisfaction

---

## Dashboard Insights
- Identified high attrition departments
- Analyzed employee satisfaction trends
- Predicted employees at risk of attrition
- Compared salary and workforce performance
- Visualized HR KPIs using interactive charts
